# A Quick Example



In order to quickly get started with the ACPred-BMF program, I will demonstrate the entire process using Main_test.fasta which can be downloaded from [Data-repository](https://github.com/RUC-MIALAB/ACPred-BMF/blob/main/Main_test.fasta) as input.

First, put the 'Main_test.fasta' into data/

```
cd ACPred-BMF
cp ../Main_test.fasta data/
```

Then, you only need to run the following order:

```
bash main.sh main
```
